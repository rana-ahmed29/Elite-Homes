# 🏠 Elite Homes

The goal of this project was to build a modern, responsive real estate landing page using Bootstrap 5, focusing on clean UI, structured layouts, and a smooth user experience.

The idea was to present real estate properties in a simple and attractive way, making it easy for users to browse listings, explore details, and get a clear visual understanding of each property.

---

## 🌐 Live Demo
🔗 [View Project](https://rana-ahmed29.github.io/Elite-Homes/)

---

## ✨ Features

- Fully responsive design (mobile, tablet, desktop)
- Dark gaming-themed UI
- Responsive navigation bar for smooth browsing
- Carousel section for featured properties
- Team showcase with hover effects
- Smooth animations & transitions
- Styled contact form
- Clean layout using Bootstrap Grid system

---

## 🧰 Built With

- HTML5 – semantic structure
- CSS3 – styling, animations, and effects
- Bootstrap 5 – responsive layout & components
- Flexbox – alignment & positioning
- Media Queries – responsiveness control
- CSS Variables – consistent design system
